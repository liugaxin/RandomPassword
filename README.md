# Random Password Generator 隨機密碼產生器

![image](https://github.com/liugaxin/randomPasswordGenerator/blob/master/pic/password.png)

## 操作說明

- 使用者可以調整產生的密碼長度
- 使用者可以選擇產生的密碼中是否包含大小寫英文字母、數字、符號在內
- 使用者可以選擇哪些字元不要出現在產出來的密碼中

## 概念

- Http/GET/POST
- Express 
- Express-handlebars
